# deep-clone

[![npm version](https://img.shields.io/npm/v/deep-clone.svg)](https://www.npmjs.com/package/deep-clone)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Fast deep clone for JSON-safe objects (no circular refs)

## Installation

```bash
npm install deep-clone
```

## Functions

- `deepClone` — Deep clone a JSON-safe value
- `deepEqual` — Deep equality check for JSON-safe values
